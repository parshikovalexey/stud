Краткое описание БД:
Таблица NOTES:
Closed – означает что данная ТТН закрыта (т.е. рассчитан вес нетто)
Returned – метка возврата ТТН (аннулирование). Если в поле указано значение 1, то обязательно будет сообщение о причине аннулирования ТТН в поле RETURNMESSAGE.
SystemGuid – уникальный индикатор организации, в которой создана ТТН. SystemGuid совпадает с одной из записи из таблицы ORGANIZATIONS
Endpoint – внутренняя организация, куда направляется груз по ТТН (id из таблицы ORGANIZATIONS). Значение 0 говорит о том, что груз направился во внешнюю организацию
ParentId – Номер ТТН на основании которой была сформирована новая ТТН (используется, когда машина приехала из пункта отпуска сырья в пункт разгрузки)
waybill – определяет тип ТТН (0 – отпуск (погрузка сырья), 1 – прием (разгрузка сырья))
Reconciliation – служебное поле, не используется
Lock – блокировка ТТН для внесения изменений
TrailerNumber – номер прицепа

Таблица WEIGHTING
В поле CONTAINERTYPE значение full определяет брутто, empty – тара.

Во всех таблицах поле Sync не используется
